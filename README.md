# giphy_assignment

A new Flutter project.

## Getting Started
A Flutter application built using Clean Architecture, BLoC state management, and the Giphy API to display trending GIFs and perform real-time GIF search with infinite scrolling.

## Frontend

Flutter (Dart)
BLoC (flutter_bloc)

## API

Giphy API
Endpoints used:

Trending: https://api.giphy.com/v1/gifs/trending
Search: https://api.giphy.com/v1/gifs/search
